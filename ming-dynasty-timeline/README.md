# 明朝历史时间线

一个交互式的明朝（1368-1644）历史时间线页面，展示16位皇帝及其统治时期的详细信息。

## 功能特点

- **竖向交互式时间线**：清晰展示明朝276年的历史脉络
- **13位皇帝信息**：包含每位皇帝的姓名、庙号、年号、统治年限和主要成就
- **可展开的详细信息**：点击卡片可查看每位皇帝的历史事件和背景
- **现代化设计**：采用优雅的配色方案，专业的视觉层次
- **平滑动画**：卡片展开、悬停效果等交互动画
- **完全响应式**：完美支持桌面、平板和手机设备

## 技术栈

- **HTML5**：语义化标签，良好的文档结构
- **CSS3**：
  - Flexbox 和 Grid 布局
  - CSS 变量（自定义属性）
  - CSS 动画和过渡效果
  - 响应式设计（媒体查询）
- **JavaScript (ES6+)**：
  - 模块化数据结构
  - 事件监听和 DOM 操作
  - 动态内容生成

## 使用方法

### 直接打开

1. 下载整个 `ming-dynasty-timeline` 文件夹
2. 用浏览器打开 `index.html` 文件
3. 开始探索明朝历史！

### 在本地服务器运行（推荐）

使用 Python 启动简单服务器：

```bash
# Python 3
python -m http.server 8000

# 或 Python 2
python -m SimpleHTTPServer 8000
```

然后访问：`http://localhost:8000`

## 自定义和扩展

### 修改配色方案

编辑 `style.css` 中的 CSS 变量：

```css
:root {
    --primary-color: #8B4513;
    --secondary-color: #D2691E;
    --accent-color: #CD853F;
    /* ... 更多变量 */
}
```

### 添加或修改皇帝数据

编辑 `script.js` 中的 `emperors` 数组：

```javascript
const emperors = [
    {
        name: '皇帝姓名',
        title: '庙号',
        era: '年号',
        reign: '开始年份 - 结束年份',
        duration: '统治时长',
        achievements: ['成就1', '成就2'],
        details: '详细信息（支持HTML）'
    }
];
```

### 调整动画速度

修改 `style.css` 中的动画延迟：

```javascript
// script.js 中
timelineItem.style.animationDelay = `${index * 0.15}s`; // 修改 0.15
```

## 历史内容说明

本页面涵盖了明朝的重要历史时期和事件：

- **建立时期**：洪武帝建立明朝，推翻元朝
- **永乐盛世**：郑和下西洋、迁都北京、《永乐大典》
- **仁宣之治**：政治清明、经济繁荣
- **中期危机**：土木堡之变、南倭北虏
- **万历中兴**：张居正改革、援朝抗日
- **王朝末期**：魏忠贤专权、农民起义、明朝灭亡

### 重要历史事件

- 靖难之役（1399-1402）
- 郑和七下西洋（1405-1433）
- 土木堡之变（1449）
- 北京保卫战（1449）
- 夺门之变（1457）
- 万历三大征（1592-1598）
- 援朝抗日（1592-1598）
- 红丸案（1620）
- 李自成攻入北京（1644）
